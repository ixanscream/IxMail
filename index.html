IxMail

ASP.NET Templating Email

Requirement

.Net 4 or later

Install using package manager console visual studio

PM> Install-Package IxMail

after installation you will see you web.config change like this : and change with your account.

<system.net>
        <mailSettings>
            <!-- Method#1: Configure smtp server credentials -->
            <smtp from="user@gmail.com">
                <network enableSsl="true" host="smtp.gmail.com" port="587" userName="user@gmail.com" password="Password" />
            </smtp>

            <!-- Method#2: Dump emails to a local directory -->
            <!--
            <smtp from="some-email@gmail.com" deliveryMethod="SpecifiedPickupDirectory">
                <network host="localhost"/>
                <specifiedPickupDirectory pickupDirectoryLocation="c:\temp\"/>
            </smtp>
            -->
        </mailSettings>
    </system.net>

Now add this class, wich this class will become your object to send your mail

// 
    public class IxMailTes
    {
        public string Name { get; set; }
        public string UserName { get; set; }
        public bool IsActive { get; set; }
    }

and run this code below :

            IxMailTes imt = new IxMailTes();
            imt.Name = "My Name";
            imt.UserName = "My User Name";
            imt.IsActive = true;

            IxWebMail IWW = new IxWebMail();
            IWW.BuildBody(imt, "yourtemplate")
                .EmailTo("yourmail")
                .EmailSubject("Activation")
                .Send();

there is 2 options method to send 1. You can send it live with .Send(). 2. Or you can send it as background process, this will be improve, when your application with big transaction SendBackground().

After you run all above, you will be see hidden directory in your project, directory address is /Views/IxMail, and you can add your template there.

example tamplate with html/txt

<html>
<head>
    <title></title>
</head>
<body>
    Hallo {{Name}} <br />
    Your user name is {{UserName}} <br />
    You ar also {{IsActive}}

    <a href="http://ixanscream.github.io/IxMail">Visit</a>
</body>
</html>

separator of {{Your Object}} would be replaced
