Ixmail

Mail ASP.NET Helper templating

View project on GitHub

Install with nuget on visual studio :

PM> Install-Package IxMail

after installation, you web config will change and add this ` <!-- Method#1: Configure smtp server credentials --> 

  <!-- Method#2: Dump emails to a local directory -->
  <!--
        <smtp from="some-email@gmail.com" deliveryMethod="SpecifiedPickupDirectory">
            <network host="localhost"/>
            <specifiedPickupDirectory pickupDirectoryLocation="c:\temp\"/>
        </smtp>
        -->
</mailSettings>

`

send mail ` using (IxMailMessage iM = new IxMailMessage()) { // an object to send iM.mailBody = new SelectListItem { Value = "Value", Text = "Text", Selected = true };

            // email to
            iM.To.Add("ix.habibi@gmail.com");

            iM.Subject = "IxMail async";

            //your tamplate
            iM.tamplateMail = "mytamplate.html";

            // your subject
            // async send mail, wich background proses, improve speed load your web
            EmailSender.SendMailAsync(iM);
        }

` Add your custom tamplates inside hidden folder IxMail, after you try, send a mail for testing :D if got something error, you can also see hidden folder in IxMail
